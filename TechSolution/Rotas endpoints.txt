Autenticação
POST /api/auth/login
body: { "email": string, "senha": string }
resp: { "token": string, "user": { "id_usuario": string, "nome": string, "email": string, "tipo": string } }
GET /api/auth/me
header: Authorization: Bearer <token>
resp: { "id_usuario": string, "nome": string, "email": string, "tipo": string }
CRUD (coleções)
Para cada recurso abaixo:

GET /api/<recurso> (lista)
POST /api/<recurso> (cria)
GET /api/<recurso>/:id (detalhe)
PUT /api/<recurso>/:id (atualiza)
DELETE /api/<recurso>/:id (remove)
Recursos:

/api/clientes
/api/ativos
/api/softwares

/api/usuarios
/api/chamados

GET /api/chamados/:id/historico
POST /api/chamados/:id/historico
GET /api/chamados/:id/anexos
POST /api/chamados/:id/anexos
